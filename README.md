# Slides Power Tools

###### _version-V1.0.1_

![logo](/assets/logo.png)

> [!NOTE]
> 這是一個基於 **WXT + React + TailwindCSS** 構建的 Chrome 擴充功能，旨在為 **Slides.com** 提供強大的增強功能。

## 簡介 & 功能

這款擴充功能專為 slide.com 的使用者設計，解決了在大量投影片中尋找內容、管理收藏以及匯出內容的痛點。

### 核心功能

- **全文模糊搜尋 (Fuzzy Search)**

  - 支援 `Cmd+K` (Mac) 或 `Ctrl+K` (Windows) 快速喚醒搜尋面板。
  - **智慧搜尋**：同時搜尋「當前簡報內容」與「已收藏的雲端投影片」。
  - **結構化預覽**：搜尋結果會標示來源（本地/收藏）、投影片頁碼，並高亮顯示關鍵字。
  - **快捷操作**：支援鍵盤上下選擇、`Enter` 跳轉、`Ctrl+Enter` 在新分頁開啟以及連按兩次 `Enter` 聚焦回投影片。
  - **流暢切換**: 透過 `ESC` 和 `Ctrl+K` 快速在投影片和搜尋框間切換，且每次按下快捷鍵會記錄當前投影片位置方便返回。

- **投影片收藏夾 (Bookmarks)**

  - **一鍵收藏**：左上角常駐按鈕，點擊即可將當前頁面加入收藏。
  - **分類管理**：支援自訂分類（Collections），方便整理不同主題的投影片。
  - **即時同步**：收藏狀態與頁面連動，切換投影片時自動更新按鈕狀態。
  - **跨裝置同步**：使用 Chrome Sync Storage，收藏資料跟著你的 Google 帳號走。
    未來將讓整個收藏內容一併同步。

- **智慧匯出 (Smart Export)**

  - **Markdown 匯出**：將整份簡報轉換為結構化的 Markdown 文件。
  - **格式還原**：完美保留程式碼區塊（含語言標記）、列表、標題層級，適合匯入 Obsidian 或 Notion 做筆記。未來將添加圖片連結抓取功能。
  - **Reveal.js 相容**：匯出的 Markdown 符合 Reveal.js 標準語法（使用 `---` 分隔），可直接轉換回投影片。

- **管理後台 (Dashboard)**
  - 提供獨立的管理頁面，可檢視所有收藏、編輯分類、批次刪除。
  - 可設定特定分類是否要在「快速搜尋」中顯示，避免搜尋結果過於雜亂。

## 安裝方法

目前擴充功能尚未上架 Chrome Web Store，請依照以下步驟手動安裝：

1.  前往本專案的 [Releases 頁面](https://github.com/jx06T/slides.com-ext/releases) (請替換為你的連結)。
2.  下載最新版本的壓縮檔（例如 `slides-ext_1.0.1.7z`）。
3.  解壓縮該檔案。
4.  打開 Chrome 瀏覽器，在網址列輸入 `chrome://extensions/` 並按下 Enter。
5.  開啟 **「開發人員模式」**。
6.  點擊 **「載入未封裝項目」**。
7.  選擇剛剛解壓縮的資料夾。
8.  完成！現在你可以打開任意 **Slides.com** 頁面開始使用。

## 使用指南

### 搜尋

- 按下 `Cmd+K` / `Ctrl+K` 開啟搜尋框。
- 輸入關鍵字，系統會自動搜尋標題、內文及程式碼。
- 點擊結果可直接跳轉至該頁。
- 上下鍵與 `Enter` 快速切換搜尋結果。
- 透過 `ESC` 關閉搜尋面板。

### 收藏

- 在投影片頁面左上角，點擊 **書籤圖示**。
- 選擇分類後按下儲存。
- 再次點擊可編輯或移除收藏。
- 雙擊 **書籤圖示** 將進入管理頁面。

### 匯出

- 點擊瀏覽器右上角的擴充功能圖示。
- 在彈出視窗中點擊 **「匯出為 Markdown」**。
- 檔案將自動下載。

## 開發者資訊

本專案使用以下技術構建：

- **核心框架**: [WXT](https://wxt.dev/) (Web Extension Framework)
- **UI 庫**: React + TailwindCSS
- **搜尋引擎**: Fuse.js
- **圖標庫**: Lucide React
